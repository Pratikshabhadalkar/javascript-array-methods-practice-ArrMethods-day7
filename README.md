# JavaScript Array Deep Methods – Day 7 Practice

This repository contains JavaScript practice for Day 7 of my MERN + AI Developer Journey.  
Today's focus was on advanced array methods like map, filter, reduce, find, sort, some, and every.

## Topics Covered
- map() – Transform array values
- filter() – Extract specific values
- reduce() – Combine array into a single value
- find() – Get first matching value
- sort() – Arrange values in order
- some() – Check if any value matches condition
- every() – Check if all values match condition

## Practice Tasks
- Double array values using map()
- Filter even numbers
- Reduce an array to a total
- Find first number greater than 50
- Sort an array in ascending order
- Check if any number is greater than 25
- Check if all numbers are positive
